Python version: 3.8.0
HOW TO USE:
run the program, enter either "1" to enter passwords or "2" to decrypt an existing file

If you chose "1":
1.enter the program you want to enter a password for (eg. Facebook)
2. Enter your username
3. Enter your password
4. Enter the password you would like to use to Encrypt/Decrypt the file with
5. Done

If you chose "2":
1. enter the file name you would like to decrypt (eg. "encryptedpasswordfor" + (program you chose) + ".txt
2. Enter the password you chose to Encrypt the file with
3. You can now view a file named "passwordfor(program you chose).txt" until you continue in the console
4. then the viewable file will then be deleted and the original encrypted file will remain.
